# pong-game
- This game is created using python turtle module.
- It is a small basic replica of most famous pong game.
- There are two paddle in the both side(moving up and down)
- A ball moving constantly in the screen and user have to restrict the ball by paddle from going outside.
- It can be played between 2 user.
- If one user misses the ball then opponent point will increases.
